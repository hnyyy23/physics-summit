# physics-summit
Physics Summit merupakan event tahunan yang diselenggarakan oleh Himpunan Mahasiswa Fisika ITS (HIMASIKA ITS) yang bergerak dalam bidang akademik dan manajerial. Kegiatan ini bertujuan untuk mengembangkan potensi generasi muda di bidang fisika serta memperluas kontribusi Departemen Fisika ITS kepada masyarakat luas.
